## 📋 Executive Summary

**Echo Split** is a novel offline mobile game scientifically designed to simultaneously train **selective attention**, **working memory**, and **inhibitory control** — the three pillars of executive function identified by neuropsychological research as most critical for daily cognitive performance.

The core mechanic has no direct equivalent in any existing app store offering, making it a genuinely new contribution to the cognitive training space.

## 01 — Research Foundation

Before designing Echo Split, extensive analysis of peer-reviewed neuropsychological research was conducted across studies from NIH PubMed, Frontiers in Neuroscience, the British Psychological Society, and meta-analyses covering 2000–2024. Below are the core scientific principles that drive every design decision.

### Key Neuropsychological Findings

#### 1. The Three Executive Functions Most Trainable by Games

Research consistently identifies three components of executive function as the highest-leverage targets for game-based cognitive training *(Miyake et al., 2000; Frontiers in Psychology, 2013)*:

- **Updating** — continuously refreshing what is held in working memory as new information arrives
- **Inhibition** — suppressing automatic, dominant responses in favor of correct but non-obvious ones
- **Shifting** — flexibly switching mental sets when task rules change

#### 2. Repetition is the Enemy of Cognitive Growth

> ⚠️ **Critical Finding (CogniFit / Neuroplasticity Research)**
>
> *"Repetition is the antithesis of cognitive training. Only by presenting the brain with NEW information and NEW challenges can we expect new neural connections to form."*
>
> Games must constantly adapt and introduce novelty to drive neuroplasticity.

#### 3. Dual-Task Training Produces Superior Results

The landmark **NeuroRacer** study *(Anguera et al., Nature, 2013)* demonstrated that a game requiring simultaneous attention to two cognitive tasks produced measurable functional changes in the prefrontal cognitive control system. These gains transferred to real-world, untrained tasks — the gold standard for effective cognitive training.

#### 4. Selective Attention and Working Memory are Deeply Linked

Research by Bavelier, Vogel, and colleagues shows that high working memory capacity individuals are fundamentally better at filtering irrelevant information before it reaches the working memory buffer. Training selective attention therefore directly improves working memory capacity — and vice versa. **This bidirectional relationship is the core mechanic of Echo Split.**

#### 5. Brain Structure Changes with Game Training

Kuehn et al. (2013) found that sustained game-based cognitive training increased gray matter density in the **right hippocampal formation** and the **dorsolateral prefrontal cortex** — brain regions critical for spatial memory and executive function.

> 📅 **Optimal dosage:** 15–20 minutes per day, 3–5 times per week

#### 6. What Makes Echo Split Genuinely New

Existing brain training apps (Lumosity, CogniFit, Peak) follow a pattern-repeat paradigm: show you something, ask you to reproduce it. Echo Split **inverts this**.

Rather than showing one stream and asking you to recall it, it shows you **TWO overlapping streams simultaneously** and asks you to mentally separate, then recall each independently. This targets a cognitive mechanism called **multi-stream selective segregation** — heavily studied in neuropsychology labs but never before gamified in a simple, accessible format.

## 02 — The Game: Echo Split

Echo Split is a simple, elegant, offline mobile game. It requires no internet connection, no in-app purchases, and no social features. Just the player and their brain.

### Core Concept

> 💡 **THE FUNDAMENTAL MECHANIC**
>
> Two streams of information are shown to the player simultaneously — one overlapping the other in time and space. The player must mentally separate the two streams and reconstruct each one independently from memory, while a brief interference task is injected between viewing and recall. Every round is procedurally generated, so no two sessions are identical.

### The Gameplay Loop

Each round follows a precise sequence grounded in working memory research:

| Phase | Name | Description |
|-------|------|-------------|
| **1** | **OBSERVE** | Two overlapping sequences of symbols flash on screen — one in **BLUE**, one in **ORANGE**. They appear at slightly different speeds and positions, interleaved in time. The player must mentally track each color stream independently. Duration: 4–8 seconds depending on difficulty. |
| **2** | **INTERFERE** | Immediately after the streams stop, a brief interference task appears — a simple but attention-demanding task for 3 seconds. This prevents simple verbal rehearsal, forcing deeper encoding. Neurologically: stresses inhibitory control and tests memory consolidation under load. |
| **3** | **RECALL** | Two separate recall panels appear — one for Blue Stream, one for Orange Stream. The player must drag symbols into the correct sequential order for each stream independently. Partial credit rewards partially correct sequences. |
| **4** | **FEEDBACK** | Instant visual feedback shows exactly where the player was right and wrong, with a memory replay at half speed. This exploits the **testing effect** — reviewing errors immediately after a test strengthens long-term encoding far more than passive re-study. |

### Why This Mechanic is Scientifically Powerful

**What it trains in the brain:**
- Selective attention (filtering Blue vs Orange)
- Dual working memory buffers (holding two parallel sequences)
- Inhibitory control (ignoring wrong-stream items)
- Sequence memory (correct ordering)
- Memory under interference (Phase 2)

**Brain regions activated:**
- Dorsolateral Prefrontal Cortex (executive control)
- Hippocampus (sequential memory encoding)
- Anterior Cingulate Cortex (conflict monitoring)
- Right Parietal Cortex (visuospatial attention)
- Cerebellum (timing and sequence learning)

## 03 — Adaptive Difficulty System

The single biggest predictor of cognitive training effectiveness is whether the task is always operating at the edge of the player's capacity. Too easy = no growth. Too hard = frustration and quitting. Echo Split uses a scientifically-calibrated adaptive engine.

### Difficulty Dimensions

| Feature | Description |
|---------|-------------|
| **Stream Speed** | Controls how fast symbols appear. Beginner: 700ms/symbol. Expert: 200ms/symbol. |
| **Sequence Length** | Starts at 3 symbols per stream. Scales to 9 per stream. Mirrors n-back task difficulty scaling from cognitive research. |
| **Symbol Similarity** | Beginner uses distinct shapes. Advanced levels introduce visually similar symbols, stressing discrimination far harder. |
| **Interference Type** | Phase 2 scales from simple (tap a color) to complex (mental arithmetic). |
| **Stream Count** | Expert mode introduces a **third stream (Green)**, demanding three-way memory segregation. |
| **Ghost Mode** | Master-level: symbols briefly appear then turn invisible mid-stream, requiring purely mental tracking. |

### The Adaptive Algorithm

After each round, the game measures accuracy and response time. If the player scores **above 80% for 3 consecutive rounds**, difficulty increases on one dimension. **Below 50% for 2 consecutive rounds**, one dimension decreases.

This keeps the player in the optimal cognitive challenge zone — the **"supply-demand mismatch" zone** that drives neuroplasticity.

## 04 — Game Modes

### 🎯 Daily Training Mode
15 minutes, 5 days/week — the research-optimal dose. The game tracks performance over 30-day cycles and shows a Cognitive Score trending graph. Each session is procedurally different from the last.
- Auto-adaptive difficulty
- Progress tracking with memory score chart
- Offline session history stored locally

### ⚡ Challenge Mode
Timed survival mode. Difficulty ramps continuously. The player continues until they make too many errors. Ideal for competitive personal bests.
- Continuous ramp-up
- Personal best tracking
- End-of-run cognitive breakdown

### 🔵 Focus Arena
Designed for ultra-long attention span training. Rounds get longer and slower. Trains the sustained attention networks studied by Cambridge's Decoder research.
- Extended session lengths (up to 30 min)
- Measures cognitive fatigue onset point

### 🌱 Kids Mode *(Ages 6–12)*
Animal-themed streams with larger, more distinct symbols. Slower pacing. Positive reinforcement on every round.
- Friendly animal characters
- Age-appropriate difficulty ceiling

## 05 — UI / UX Design Principles

### Design Philosophy

> The interface must **disappear**. Every pixel of UI that the player has to decode is cognitive load stolen from the actual training task.

Echo Split uses a brutally minimal aesthetic: dark background, only two or three colors on screen at any time, no text during gameplay phases, and audio cues that are distinct but never jarring.

| Element | Specification |
|---------|--------------|
| **Color Coding** | Blue = Stream A, Orange = Stream B, Green = Stream C. Colorblind-safe pairs. Background is deep navy `#0F0F1A`. |
| **Symbol Library** | Geometric shapes only — never letters or numbers. This prevents verbal rehearsal strategies that bypass visuospatial WM training. |
| **Audio Design** | Stream A: soft bell tone. Stream B: soft chime. Correct = ascending tone. Error = neutral tone (never harsh). |
| **Session Start** | 10-second breathing prompt before each session. Research shows even brief pre-task mindfulness measurably improves WM by reducing cortisol load. |
| **Progress Visualization** | Radar chart: Speed, Accuracy, Stream Separation, Sequence Order, Interference Resistance. |

## 06 — Technical Architecture

### Recommended Stack for AI-Assisted Development

| Layer | Technology |
|-------|-----------|
| **Platform** | React Native (Expo) — single codebase for iOS + Android. Offline-first. |
| **Offline Storage** | SQLite via `expo-sqlite` — all sessions stored locally. No server required. |
| **Stimulus Engine** | Custom `EchoStreamEngine` TypeScript class. Procedurally generates unique sequences using seeded PRNG. |
| **Animation** | React Native Reanimated 3 — GPU-accelerated for precise millisecond stimulus timing. |
| **AI Integration** | Optional: local on-device LLM (`llama.rn`) for personalized difficulty recommendations. Fully offline. |
| **Scoring** | Composite **Cognitive Performance Index (CPI)**: Accuracy (40%) + Speed (25%) + Stream Separation (25%) + Interference Resistance (10%). |

### Data Model

```sql

sessions (
  session_id, date, mode,

rounds (
  round_id, session_id,

  stream_a_recalled, stream_b_recalled,
  accuracy, response_time_ms
)

player_profile (
  player_id, skill_level,
  current_difficulty_json,
  30_day_cpi_history_json
)
```
